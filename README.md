## ⚡ Quick Start (Workshop Guide)

### 1. Configure the Core
Copy the blueprint:
```bash
cp .env.example .env
```
Fill in your encrypted keys (LinkedIn Token, Organization URN, Gemini API Key).

### 2. Set the Manifest
The `content_manifest.json` is your JARVIS command center. Set your dates, times, and captions here.

### 3. Initialize "Cloud Mode"
Push this code to GitHub and set your **Repository Secrets**. The `social_media_post.yml` workflow will handle the rest—even while you're asleep.

```bash
# Set the auto-pilot schedule
# Fires at exactly 7:00 PM IST (13:30 UTC) daily.
cron: "30 13 * * *"
```
